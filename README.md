# Movie Library

## Автор
**Шамсиева Фарангис** 

## Описание
Приложение для хранения информации о фильмах с графическим интерфейсом (Tkinter).  
Позволяет:
- Добавлять фильмы с названием, жанром, годом выпуска и рейтингом.
- Просматривать список фильмов в таблице.
- Фильтровать по жанру и году выпуска.
- Удалять выбранные фильмы.
- Автоматически сохранять и загружать данные в JSON-файл.

## Валидация данных
- Год — только целое число.
- Рейтинг — число от 0 до 10 включительно.
- Все поля обязательны для заполнения.
